🚀 Lambetree - Advanced Link Management System
Lambetree adalah aplikasi link management (Linktree Clone) yang dibangun dengan fokus pada Real-time Sync dan Premium UI/UX. Proyek ini dirancang untuk memberikan pengalaman pengelolaan tautan yang mulus dengan simulasi pratinjau perangkat seluler secara langsung.

Jelajahi

✨ Fitur Utama
Real-time Dashboard: Sinkronisasi instan antara editor admin dan halaman publik menggunakan Firebase Firestore.

Live Mobile Preview: Simulasi tampilan HP di dalam dashboard untuk melihat perubahan secara langsung.

Smart Icon Detector: Otomatis mendeteksi dan menampilkan ikon brand (Shopee, Tokopedia, WhatsApp, dll) berdasarkan judul tautan.

Premium UI/UX: Desain modern dengan sistem Flexbox yang responsif, mencegah teks tumpang tindih (anti-clashing layout).

Secure Authentication: Sistem login admin yang aman menggunakan Firebase Auth.

Custom Branding: Skema warna yang diselaraskan dengan identitas brand (Red "L" Signature Style).

🛠️ Tech Stack
Frontend: HTML5, CSS3 (Custom Variables & Flexbox), Modern JavaScript (ES6+).

Backend: Firebase Firestore (NoSQL Database).

Auth: Firebase Authentication.

Icons: Font Awesome 6.0+ Pro Icons.

Fonts: Inter (Google Fonts).
